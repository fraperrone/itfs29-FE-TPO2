# Proyecto React + Vite

Este repositorio contiene una aplicación desarrollada con React y Vite.

## 📁 Estructura del proyecto

El árbol de componentes y archivos se encuentra documentado en este mismo directorio, en el archivo:

DOCUMENTACION - estructura_y_arbol_react

Allí se detalla la organización de carpetas, componentes reutilizables, vistas y lógica compartida.

## 🚀 Scripts disponibles

- `npm run dev` — Inicia el servidor de desarrollo.
- `npm run build` — Genera el build de producción.
- `npm run preview` — Previsualiza el build localmente.

## 🧩 Tecnologías principales

- React
- Vite
- Axios
- React Router

## 📦 Instalación

```bash
npm install
npm run dev
